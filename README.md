# CardioFatigue AI

A smartphone PPG and symptom-based screening app for early assessment of cardiovascular-related fatigue and autonomic burden.

## Features
- User login / signup
- User ID generation
- Symptom-based screening
- Smartphone PPG-based signal input
- Signal quality checking
- Explainable AI output
- Weekly history map
- Doctor suggestion
- Wellness follow-up

## Tech Stack
- Python
- Streamlit
- SQLite
- NumPy
- Pandas
- scikit-learn
