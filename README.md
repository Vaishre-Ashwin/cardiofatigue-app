🫀 CardioFatigue AI

A smartphone PPG and symptom-based screening application for early assessment of cardiovascular-related fatigue and autonomic burden.

This system combines biomedical signal processing, machine learning, and AI-based explanation to provide preventive health insights, not diagnosis.

---

📌 Overview

CardioFatigue AI is designed as a low-cost, accessible screening tool that uses:

- 📱 Smartphone-based PPG signal input
- 🧾 Symptom-based inputs
- 🧠 Machine learning analysis

to estimate physiological stress and cardiovascular fatigue risk.

«⚠️ This application is intended for screening and monitoring only, not for medical diagnosis.»

---

🚀 Features

🔹 Core Features

- 🔐 User login / signup
- 🆔 Unique user ID generation
- 🧾 Symptom-based screening
- 📱 Smartphone PPG signal input
- 🧠 ML-based risk prediction
- 🩺 Doctor suggestion system
- 🧘 Wellness follow-up guidance

---

🔹 Advanced Features (NEW 🔥)

- 🔍 Signal Quality Check (SQI)
  
  - Ensures reliable PPG input using noise, peak, and consistency checks

- 📊 Temporal Trend Analysis
  
  - Tracks user history and detects changes over time

- 💡 Explainable AI Output
  
  - Provides clear reasons like:
    «“Low HRV, high stress, elevated heart rate”»

- 🤖 Gemini AI Assistant Integration
  
  - Converts results into human-friendly explanations
  - Provides safe health guidance
  - Answers user queries

---

🧠 System Workflow

User Input (PPG + Symptoms)
        ↓
Signal Quality Check
        ↓
PPG Feature Extraction
        ↓
Machine Learning Model
        ↓
Trend Analysis (History)
        ↓
Explainability Module
        ↓
Gemini AI Assistant
        ↓
Final Output (Risk + Explanation + Advice)

---

⚙️ Tech Stack

- Frontend: Streamlit
- Backend: Python
- Database: SQLite
- Machine Learning: scikit-learn
- Signal Processing: NumPy, SciPy
- Visualization: Matplotlib
- AI Assistant: Google Gemini API

---

📊 Output

The system provides:

- ✅ Risk Level (Low / Moderate / High)
- 📈 Risk Score
- 💡 Explanation (why result came)
- 🤖 AI-generated guidance
- 📊 Weekly trend visualization
